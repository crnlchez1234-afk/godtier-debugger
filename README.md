## 📋 Descripción

Este proyecto ha sido unificado y limpiado para separar la funcionalidad real de los componentes experimentales o incompletos.
Integra un sistema de **Auto-Debugging** basado en análisis estático (AST) con un núcleo de **IA Neuro-Simbólica** (NeuroSys AGI) para razonamiento avanzado.

## 📁 Estructura del Proyecto

`
DEBUGGING_GODTIER/
│
├── main.py                  # Punto de entrada principal
├── src/                     # Código fuente unificado
│   ├── ai/                  # Modelos de IA (NeuroSys Core)
│   └── debugger/            # Lógica de Auto-Debugging
├── requirements.txt         # Dependencias reales
└── README.md                # Documentación
`

## 🚀 Inicio Rápido

1.  **Instalar dependencias:**
    `ash
    pip install -r requirements.txt
    `

2.  **Ejecutar el sistema:**
    `ash
    python main.py
    `

## 🔧 Componentes

### 1. Auto-Debugger (src/debugger)
Sistema capaz de analizar código Python, detectar errores de sintaxis y patrones comunes de fallo, y aplicar correcciones automáticas.

### 2. NeuroSys AI (src/ai)
Núcleo de inteligencia artificial neuro-simbólica que combina redes neuronales (PyTorch) con grafos de conocimiento para un razonamiento más profundo sobre el código.
