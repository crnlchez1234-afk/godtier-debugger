## Quick Start

1. Install the base package:
   pip install godtier-debugger

2. Unzip this PRO bundle into your project

3. Run the Aurora setup:
   python aurora_setup/install.py

4. Use PRO features:
   godtier analyze myfile.py    # Now uses AI instead of heuristics
   godtier dashboard             # Live monitoring
