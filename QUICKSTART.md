## 🚀 Activación del Sistema (3 Pasos)

### 1️⃣ Clonar el repositorio (si aún no lo tienes)
```powershell
git clone <tu-repo>
cd DEBUGGING_GODTIER
```

### 2️⃣ Activar el sistema
**Windows (PowerShell):**
```powershell
.\ACTIVATE.ps1
```

**Linux/Mac:**
```bash
chmod +x ACTIVATE.sh
./ACTIVATE.sh
```

### 3️⃣ ¡Listo! Ya puedes usar el sistema

## 💡 Comandos Básicos

### 🔍 Auto-Debugging
Detecta y corrige errores automáticamente:
```bash
python main.py debug mi_script.py
```

### 🧬 Optimización Darwin
Optimiza el rendimiento de tu código:
```bash
python main.py optimize mi_script.py
```

### ♻️ Auto-Reparación Lazarus
Repara código que falla en tiempo de ejecución:
```bash
python main.py heal mi_script.py
```

### 📊 Dashboard de Auditoría
Genera un reporte completo del proyecto:
```bash
python main.py audit
```

### 🧪 Ejecutar Tests
```bash
pytest tests/ -v
```
